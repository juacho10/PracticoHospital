# Sistema de Información Hospitalaria (HIS)

Aplicación web para la gestión de admisiones y atención de pacientes en un hospital.

## Requisitos

- Node.js (v14 o superior)
- MariaDB/MySQL
- XAMPP (para la base de datos local)
